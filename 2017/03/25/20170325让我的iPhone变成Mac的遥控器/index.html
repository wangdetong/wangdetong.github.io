让我的iPhone变成Mac的遥控器

Posted on 2017-03-25 | In Mac

概要

背景

有一次我刚回到家，想起来公司的Mac 没有关机，由于最近公司强调保护自己隐私与节约用电，需要在下班后Mac关机，我就着急忙慌的给加班的同事打电话，帮忙给我的Mac关机；
有时，晚上我躺着床上用Mac看电影，看困了，还不得不爬起来去关电脑；
💡如果我们可以iPhone远程控制Mac，就可以解决上面的问题了。💡

原理

Mac 上需要有一个能根据iPhone上的指令自动触发 AppleScript 的后台程序。而正好，Mac 上的 Mail & Message 都自带了自动运行 AppleScript 功能

方案

根据Mail 或 Message收到的信息，去执行 AppleScript 来控制Mac

mail篇

mail让Mac休眠

通过我的QQ邮箱给 Mac上的邮箱发邮件，达到屏幕休眠的效果。

步骤

1，mail 启用自动触发 AppleScript 的方法是在应用偏好设置:「规则 - 添加规则」

2，填写内容

  1. 描述：QQ Close Screen
  2. 执行脚本的条件：（避免他人给自己发邮件，造成误伤！）
    发件人:1875734581@qq.com
    主题:CloseScreen
  3. 满足条件后执行的事件:
    删除邮件、标记已读（毕竟是自己发给自己的邮件，可以保留）
    执行脚本：选择自己的脚本

3，创建脚本 Close Screen.scpt，其内容是：
do shell script "pmset displaysleepnow"
点击运行，屏幕会休眠。

存储到mail指定的文件夹下，然后在规则中选择Close Screen.scpt

拓展

现在通过mail可以让屏幕会休眠了，当然修改命令 可以到达 Mac锁屏／关机的效果

message篇

message让Mac休眠

Message 启用自动触发 AppleScript 的方法是在应用偏好设置中打开 「通用 - AppleScript 处理程序」菜单，从中选择我们想要运行的 AppleScript。

令人意外的是，苹果已经为我们准备好了几个十分有用的 AppleScript，例如用来遥控 iTunes 的 iTunes Remote Control，现在打开iTunes Remote Control.applescript

iTunes Remote Control.applescript是既然是控制 iTunes 的，我们需要写一个控制Mac Screen的，则可以copy一份applescript并命名为PlayGame.applescript，只需要修改方法on message received theMessage from theBuddy for theChat就行了。

测试一下：自己发送一个”CloseScreen;” Mac休眠了。

mail & message 简单比较

mail

简单：基本不需要编程，只需要配置执行脚本的条件就行了
机型：iPhone与android手机，只要按照任意一款mail.app,比如QQ邮箱…

message

灵活：需要编程来实现（可以扩展更多功能）
机型：iPhone手机（iMessage）
