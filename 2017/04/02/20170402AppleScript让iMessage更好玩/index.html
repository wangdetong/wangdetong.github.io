iMessage的扩展

实战：自动回复机器人

当朋友给我发iMessage消息时，提示其我的功能列表（现在只有menu,shell,about功能）；
当他选择某一功能后，我的机器人自动返回他想要的信息；
如果朋友输入错误指令，有错误提示；
如果长时间不回复，有超时提示；
如果朋友不想玩了，有退订功能。

效果




简要设计：

  • menu shell about、TD 关键字，可以把这四个关键字比如称四个目录。无论sender当前在哪个目录下，只要回复任一个目录都需要跳到对应的目录下，比如当前在 menu目录下，回复shell 则进入shell 目录
  • senderFile:记录sender当前进入的目录，比如menu,sender只能回复menu下的指令，比如 “3”，如果回复 “ls -al” 则弹出无效。
  • menu：展示列表，输入指令代码，返回结果
  • shell：过滤掉已知的危险命令，然后通过do shell script 执行 message
  • about：返回about信息
  • TD or timeout：清空之前的记录，回到主目录
  • log：记录用户的操作（永久记录）
